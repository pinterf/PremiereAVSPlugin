Premiere AVISynth Plugin v1.96 - Source release

This is an update of the Premiere Avisynth Import Plugin originally written by Ben Rudiak-Gould. It allows Adobe's Premiere video editing package to read Avisynth AVS scripts.

It is compatible with Premiere 5.x, 6.x and Premiere Pro 1.x, 2.x, CS3, CS4 and Avisynth 2.6 or Avisynth+.

The homepage for the plugin is http://videoeditorskit.sourceforge.net, kindly hosted by Sourceforge. Check the webpage for new releases and full source.

Plugin was updated in 2020, project moved to github (pinterf)

Building Instructions

Premiere Plugin

In order to build you will require the following things:

  • Build environment is (as for 1.96 release) Visual Studio 2019
  • Project moved to github (pinterf)
  • The headers from the Premiere 6.5/Pro SDK placed in Source/premiere-headers (or somewhere in the include path)
  • Download SDK from https://www.adobe.com/devnet/premiere/sdk/cs3.html#
  • As of 2020, actual SDKs (2015-) no longer contain necessary headers for a successful build

The plugin should build with no errors or warnings.

GUI

The GUI may be built using NSIS version 2. Just run 'makensis.exe PremiereAVSGUI.nsi'.
